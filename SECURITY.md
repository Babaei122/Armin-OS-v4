# Security Policy

## گزارش آسیب‌پذیری

جزئیات آسیب‌پذیری را در Issue عمومی منتشر نکنید. ابتدا از روش تماس خصوصی مالک مخزن در GitHub استفاده کنید و نسخه مرورگر، مراحل بازتولید و اثر احتمالی را بنویسید.

## محدوده امنیت

PlanOS یک برنامه local-first است. PIN فعلی قفل حریم خصوصی است و داده‌های ذخیره‌شده روی دستگاه را رمزگذاری نمی‌کند. از ذخیره اطلاعات بسیار حساس یا اسرار سازمانی در برنامه خودداری کنید و از داده‌ها خروجی پشتیبان بگیرید.
