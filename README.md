# PlanOS (Armin OS v4)

یک برنامه‌ریز شخصی فارسی و راست‌به‌چپ برای مدیریت تسک‌ها، پروژه‌ها، اهداف، یادداشت‌ها، دارایی‌ها و تقویم جلالی است. برنامه به‌صورت PWA اجرا می‌شود و داده‌ها را روی مرورگر کاربر نگه می‌دارد.

## قابلیت‌های برنامه‌ریزی

- هدف، پروژه و تسک به هم متصل‌اند و وضعیت کارهای مرتبط در هر بخش دیده می‌شود.
- Inbox برای ثبت سریع کارهای بدون تاریخ و تعیین‌تکلیف بعدی وجود دارد.
- «فرمان امروز» ظرفیت روز، زمان برنامه‌ریزی‌شده، هشدار بیش‌ازظرفیت و سه اولویت اصلی را نشان می‌دهد.
- موارد عقب‌افتاده را می‌توان مستقیم انجام داد یا به امروز/فردا منتقل کرد.
- مرور هفتگی، کارهای تکمیل‌شده، Inbox، عقب‌افتادگی و پروژه‌های بدون اقدام بعدی را جمع‌بندی می‌کند.
- بینش روزانه، تسک‌ها و پروژه‌های عقب‌افتاده را پیدا می‌کند و اقدام مستقیم پیشنهاد می‌دهد.
- جملهٔ دیسیپلین روزانه به‌صورت خودکار و ثابت برای هر روز انتخاب می‌شود.
- هر تسک می‌تواند ساعت یادآوری، اعلان زودتر و تکرار روزانه، هفتگی یا ماهانه داشته باشد.
- اعلان مرورگر هنگام اجرای برنامه بررسی می‌شود و یادآورهای ازدست‌رفته پس از بازگشت به برنامه نمایش داده می‌شوند.
- خروجی `ICS` هر تسک را می‌توان در تقویم گوشی یا لپ‌تاپ باز کرد تا اعلان بومی دستگاه حتی هنگام بسته‌بودن PlanOS فعال باشد.
- تقویم در موبایل نمای فشرده، خلاصهٔ روز، فهرست برنامه و هشدار موارد عقب‌افتاده دارد.

## اجرا

به یک سرور استاتیک نیاز است؛ برای نمونه:

```bash
python -m http.server 8000
```

سپس `http://localhost:8000` را باز کنید. بازکردن مستقیم `index.html` برای Service Worker مناسب نیست.

## داده و حریم خصوصی

- داده‌های اصلی در مرورگر ذخیره می‌شوند و به حساب ابری متصل نیستند.
- از بخش «بیشتر» مرتب خروجی JSON بگیرید.
- پیش از Import و Reset یک بکاپ ایمنی در IndexedDB ایجاد می‌شود.
- PIN یک قفل حریم خصوصی برای جلوگیری از مشاهده اتفاقی است؛ امنیت سازمانی یا حفاظت در برابر فردی که به فایل‌های پروفایل مرورگر دسترسی دارد فراهم نمی‌کند.
- قیمت رمزارز و نرخ تبدیل از APIهای عمومی دریافت می‌شود.

## توسعه و تست

```bash
npm test
```

این بررسی syntax جاوااسکریپت، شناسه‌های تکراری، اتصال labelها، metadata مربوط به PWA و چند الزام امنیتی را کنترل می‌کند. گردش‌های اصلی باید پیش از merge روی دسکتاپ و موبایل نیز آزمایش شوند.

## ساختار

- `index.html`: رابط، styleها و منطق فعلی برنامه
- `service-worker.js`: cache و پشتیبانی آفلاین
- `manifest.webmanifest`: تنظیمات نصب PWA
- `scripts/validate.mjs`: کنترل‌های استاتیک و regression

## سیاست تغییر

فرمت کلید ذخیره‌سازی `planos_v1` برای سازگاری نسخه‌های قبلی ثابت نگه داشته شده است. schema نسخهٔ ۵ ارتباط هدف/پروژه/تسک، Inbox و مرور هفتگی را با migration سازگار اضافه می‌کند.

## مجوز

این مخزن فعلاً فایل مجوز ندارد؛ تا زمان انتخاب مجوز توسط مالک، حقوق استفاده مجدد به‌طور خودکار اعطا نمی‌شود.
